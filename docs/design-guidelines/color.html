Color and Typography

On a Television color helps to indicate interactivity. On a smartphone or computer the user uses his finger or cursor to navigate inside the application. On a television we dont have this indicator, so the application needs a clear focus color to show the users location.
Recommended
Not Recommended
Important note: if you create a new application, make sure the base colors include a high contrast color that indicates the location (focus) on screen. Consider choosing a key color to indicate interactivity and state.

Do not use the color white (#FFFFFF)

Clear white pops out of the television screen. Instead of using clear white, its better to use off-white (#F1F1F1).

Font size matters

Because of the high distance between the user an his television text should always be legible. Text should nevver be smaller than - 24 pixels / 18 points.

Line height

Using a line height of 125% or larger makes text (better) readable from a larger distance.

Font usage

In general, MAF applications use a single font. Instead of mixing fonts, it is better to mix different font sizes and weights.

Text and image alignment

For a clean design it's important to align the text and images by using for example a simple grid structure.
