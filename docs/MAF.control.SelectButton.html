MAF.control.SelectButton

When selected by the user this will create a new view on which the user can select one of the options of this component. The view created is default MAF.system.OptionSelectView

Needs to be referenced on the this.controls object of the view. Also needs a guid, otherwise the states will not be stored between the views.
